Verkefni 1

Að búa til vefsíðu með HTML


Fyrir þetta verkefni varð að búa til nothæfa heimasíðu sem hver sem er hefði aðgang að. Kröfur verkefnisins voru að vefsíðan yrði að vera forrituð með HTML5.
Langt var síðan ég vann síðast með HTML svo byrjað var á því að sækja "template" af HTML5 UP sem hýsir fullt af ókeypis "template-um".

Á HTML5 UP fann ég svo þema sem ég var sáttur með, "downloadaði" því og gat ég þá farið að breyta því til þar til ég var sáttur.
Til þess að forrita í HTML þarf að nota "compiler" og þar sem ég er vanur að forrita allt með Visual Studio Code vildi ég halda áfram að nota það.

Til þess að forrita HTML5 í vscode varð ég að fá mér Live Preview "plugin-ið" og eftir að hafa sett það upp gat ég byrjað að forrita HTML í vscode.

Til þess að sýna verkefnin ákvað ég að notast við modal image sem opnar glugga þegar smellt er á mynd. Þannig gat ég haft öll verkefnin á sömu síðu sem gerði allt meira "clean".
Með því að nota modal myndir gat ég búið til slideshow gallery sem leyfði mér að skipta þægilega á milli verkefna á sömu síðu.
Til að byrja með voru örvarnar á hliðunum fastar á síðunni en með hjálp stack overflow gat ég látið þær fylgja með skjánum þegar skrollað var niður.
Þegar síðan var tilbúin sótti ég Git sem er forkrafa þess að hlaða síðunni upp á GitHub.

Þegar búið er að sækja git þarf að hlaða síðunni upp á github.
Í fyrstu reyndi ég að hlaða síðunni upp á github með notkun GitBash en þar sem það virkaði illa sótti ég GitHub Desktop sem reyndist mun auðveldara að nota.

Þegar búið er að sækja GitHub Desktop verður að skrá sig inn og þegar það er komið er hægt að velja "Create a New Repository on your hard drive..."

Síðan þarf að skýra svæðið og stilla "Local path" til þess að vera það sama og mappan sem HTML skráin fyrir síðuna er geymd í.

Þegar búið er að færa síðuna í möppuna verður að skrifa lýsingu í "Summary (required)" reitinn og þar næst er hægt að smella á "Commit to main" og að lokum "Push origin" sem hleður síðunni upp á GitHub.

Þá skal förinni heitið á GitHub og svo á repository síðu verkefnisins. Þá skal velja "Settings" og svo "Pages". Þar undir "Source" skal "Branch:" vera stillt á "main" og síðan smella á "Save". Þá birtist linkur að heimasíðunni.
