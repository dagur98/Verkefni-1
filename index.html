Verkefni 1

Að búa til vefsíðu með HTML


Fyrir þetta verkefni varð að búa til heimasíðu með HTML5. Kröfur verkefnisins voru einfaldar, vefsíðan skyldi skilvirk og auðvelt að nota, gerð af mér og aðgengileg hverjum sem er.
Áður en ég gat hafist handan varð ég fyrst að ákveða forritið sem síðan yrði forrituð í sem og forritið Git sem var nauðsynlegt til þess að hlaða síðunni á netið.
Þar sem ég forrita að mestu í Visual Studio Code og vildi halda því áfram fyrir þetta verkefni sótti ég viðbótina Live Preview sem leyfir Visual Studio Code að forrita í HTML, CSS og Javascript.

Þar næst sótti ég Git og hlóð því á tölvuna.

Til þess að hýsa síðuna á netinu var notast við síðuna GitHub sem býður upp á ókeypis hýsingu fyrir litlar vefsíður.

Til þess að hlaða síðunni á GitHub var annars vegar hægt að nota Git Bash en þar sem það virkaði illa sótti ég GitHub Desktop forritið sem auðveldaði allt ferlið.

Þegar búið er að sækja GitHub Desktop verður að skrá sig inn eða ef ekki er búið að búa til aðgang, gera það fyrst og skrá sig svo inn.

Þá skal velja "Create a New Repository on your hard drive..." sem býr til möppu á tölvunni sem tengist GitHub. Hægt er að vista möppuna hvar sem er á tölvunni en mælt er með því að búa hana til undir GitHub möppunni í "Documents".
Möppunni skal vera gefið nafn og lýsing og svo skal velja "Create repository".

Þá er búið að búa til möppu tengda GitHub og þá má byrja á heimasíðunni.
Þar sem ég hafði lítið unnið með HTML fyrir þetta verkefni byrjaði ég á að sækja sniðmát af HTML5 UP sem hýsir ókeypis sniðmát sem hver sem er getur tekið og breytt eftir sínum þörfum

Á HTML5UP fann ég sniðmátið "astral" sem ég var sáttur við, sótti það og byrjaði að breyta því.

Ég hlóð niður sniðmátinu og færði það í möppuna sem ég hafði búið til áður. Þegar .zip skráin var komin þangað valdi ég "extract here" sem hlóð niður öllu því sem ég þurfti til að byrja á heimasíðunni.
Ég hægri smellti þá á index skránna og valdi "Open with Visual Studio Code".

Með því að nota Modal myndir gat ég búið til Slideshow gallery sem leyfði mér að skipta þægilega á milli verkefna innan sömu síðunnar.
Til að byrja með voru örvarnar sem færðu á milli verkefna fastar á síðunni en með hjálp stack overflow tókst mér að stilla þær svo þær fylgdu eftir þegar síðan var færð niður.
Fyrir ferilskránna sótti ég sniðmát af SDR og lagaði til svo það passaði á síðuna mína.
Þegar síðan var tilbúin opnaði ég GitHub Desktop, fyllti í "Summary (required)" reitinn, valdi "Commit to main" og að lokum "Publish repository". Þá var ég búinn að hlaða síðunni upp.

Til þess að finna hlekkinn að síðunni opnaði ég GitHub í vafra, opnaði verkefnið, valdi "Settings", "Pages" stillti "Branch:" á "main" og að lokum smellti ég á "Save". Við það birtist hlekkur sem tekur mann á síðuna.

Þá var síðan tilbúin.
