Verkefni 1

Að búa til vefsíðu með HTML


Fyrir þetta verkefni átti að búa til heimasíðu með HTML5.
Kröfur verkefnisins voru þær að vefsíðan varð að vera skilvirk og auðveld í notkun. Hún átti að vera búin til af mér og aðgengileg hverjum sem er.
Áður en ég gat hafist handa varð ég að ákveða hvaða forrit ég ætti að nota.
Þar sem ég forrita að mestu í Visual Studio Code vildi ég halda því áfram svo ég sótti mér viðbótina Live Preview. Viðbótin leyfir Visual Studio Code að forrita í HTML, CSS og Javascript.


Þar næst sótti ég Git sem leyfir mér að hlaða síðunni á netið.


Til þess að hýsa síðuna á netinu notaði ég GitHub sem býður upp á ókeypis hýsingu fyrir litlar vefsíður.


Til þess að hlaða síðunni á GitHub var annars vegar hægt að nota Git Bash en þar sem það virkaði illa sótti ég GitHub Desktop forritið. Þetta forrit er mun notandavænna og þægilegra í notkun.


Þegar búið er að sækja GitHub Desktop þá þarf að skrá sig inn eða búa til aðgang á GitHub.


Þegar búið er að skrá sig inn á GitHub Desktop skal velja "Create a New Repository on your hard drive...". Eftir að búið er að fylla í reitina skal velja "Create repository" sem býr til möppu þar sem innihald síðunnar verður geymt. Þessi mappa tengist GitHub í gegnum netið.


Að því loknu er hægt að byrja á vefsíðugerðinni.
Þar sem ég hef lítið unnið með HTML fyrir þetta verkefni byrjaði ég á að sækja sniðmát af vefsíðunni HTML5UP. Þessi vefsíða hýsir ókeypis sniðmát sem hægt er að breyta eftir þörfum.


Á HTML5UP fann ég sniðmátið Astral sem ég hlóð niður og byrjaði að breyta eftir mínum þörfum.


Eftir að hafa hlaðið niður sniðmátinu fékk ég .zip skrá sem ég færði í möppuna sem ég hafði áður búið til. Þegar skráin var komin þangað valdi ég "extract here" sem hlóð niður öllu því sem ég þurfti til að byrja á heimasíðunni.
Því næst hægri smellti þá á index.HTML skránna og valdi "Open with Visual Studio Code".


Með því að nota Modal myndir gat ég búið til Slideshow gallery sem leyfði mér að skipta þægilega á milli verkefna innan sömu síðunnar.
Myndirnar efst á síðunni komu frá Font Awesome sem býður upp á hundruði mynda mynda eins og t.d. mappan eða GitHub myndin efst sýnir. Kosturinn við Font Awesome myndirnar er að auðvelt var að nota þær sem takka á síðum, láta þær blikka með CSS kóða eða breyta um lit með Javascript.
Notast var við audio eiginleikann í HTML til þess að spila .mp3 skrá á síðunni.
Til að byrja með voru örvarnar sem færðu á milli verkefna fastar á síðunni en með hjálp Stack Overflow tókst mér að stilla þær svo þær fylgdu eftir þegar síðan færðist upp og niður.
Fyrir ferilskrána sótti ég sniðmát af SDR og lagaði til þar til hún passaði á vefsíðuna.
Þegar síðan var tilbúin opnaði ég GitHub Desktop, fyllti í "Summary (required)" reitinn, valdi "Commit to main" og að lokum "Publish repository". Þá var ég búinn að hlaða síðunni upp á GitHub.


Til þess að finna hlekkinn að síðunni opnaði ég GitHub í vafra og opnaði verkefnið. Síðan valdi ég "Settings" og þar næst "Pages". Þegar þangað var komið stillti ég "Branch:" á "main" og að lokum smellti ég á "Save". Við það birtist hlekkur sem flytur mann á síðuna.


Þá var síðan tilbúin og verkefninu lokið.
